Packagecom.adobe.xml.syndication.generic
Classpublic class RSS20Item
ImplementsIItem

Class that abstracts out the specific characteristics of an RSS 2.0 item into a generic Item. You create an instance using an Item20 object, then you can access it in a generic way.
