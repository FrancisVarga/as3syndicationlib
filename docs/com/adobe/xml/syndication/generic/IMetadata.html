Interfacepublic interface IMetadata
ImplementorsAtom10Metadata, RSS10Metadata, RSS20Metadata

Interface that defines the characteristics common across Atom and RSS metadata. Metadata is defined as data relating to the feed itself as opposed to the feed's entries or items.
