dateCheck()method
public static function dateCheck(x:XMLList, f:Function):Date

Checks to see if a piece of XML that should be a date is empty. If so, returns null. If not, parses the string into a date using the specifid date parsing function.

Parameters
x:XMLList — The piece of XML to test.
 
f:Function — The date parsing function to use to parse the string if it's not null.

Returns
Date — null if the date is emply, otherwise a date object created using the specified date-parsing function.
